#### 강화학습(Reinforcement Learning, RL))
# 에이전트 (Agent)	학습을 수행하는 주체. 예: 로봇, 게임 플레이어
# 환경 (Environment)	에이전트가 상호작용하는 세계. 예: 게임, 시뮬레이션
# 상태 (State)	현재 환경의 정보. 예: 체스판의 말 배치
# 행동 (Action)	에이전트가 선택할 수 있는 행동. 예: 말을 움직이기
# 보상 (Reward)	행동의 결과로 받는 점수. 예: 승리 시 +1, 패배 시 -1
# 정책 (Policy)	어떤 상태에서 어떤 행동을 할지 결정하는 전략
# 가치 함수 (Value Function)	특정 상태나 행동이 얼마나 좋은지를 평가

### 학습과정
# 1. 에이전트가 환경에서 상태를 관찰함
# 2. 정책에 따라 행동을 선택함
# 3. 환경이 그 행동에 대한 보상을 주고, 새로운 상태를 반환함
# 4. 에이전트는 이 정보를 바탕으로 정책을 개선함
